== Description ==
The Singlepagefolio theme for WordPress is a fully responsive theme that looks great on any device, even with a featured small devices version. Is a very unique web portfolio, in a single page application. Has a blog also with features, including a front page template with its own widgets if you want, styling for post formats and an optional no-sidebar page template. Make it yours with custom colors, background image, and background color.


== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Search the .zip file you have downloaded and click the singlepagefolio theme .zip file.
3. Click on the 'Activate' button to use your new theme right away.
4. Activate the request plugins.
5. Import the demo data.
6. Go to documentation folder to learn how to use it and customize it.
7. Load your content and enjoy it.
